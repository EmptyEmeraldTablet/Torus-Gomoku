# 环面五子棋（单棋盘版）

在线示例：https://torus-gomoku.pages.dev/

**游戏规则**
- 棋盘为环面拓扑：上下、左右边界相连。
- 棋子落在网格交叉点上，黑先白后轮流落子。
- 任意方向（横、竖、斜）形成连续五子即获胜，连线允许跨越边界。
- 右边与下边为虚线提示边，落子会自动映射到对应的实线边交点。
- 支持拖动平移视图，查看环面位置关系。

**当前开发情况**
- 已实现：棋盘渲染、环面坐标转换、胜负判定、拖动平移、重置、撤回、棋盘大小切换、落子自动包裹提示动画、可落子范围提示、状态栏显示。
- 待完善：移动端触控体验细化、性能与交互细节优化、自动测试。

**本地开发**
- 依赖：Node.js + npm
- 启动开发环境：
  - `npm install`
  - `npm run dev`
